Taller 2 - Ciencia de Datos Aplicada

Este repositorio contiene la solucion al Taller 2 del curso de Ciencia de Datos Aplicada de MINE 2025-02. El objetivo es evaluar las capacidades del estudiante para aplicar técnicas de machine learning, las cuales permita descubrir insights, para predecir el precio de venta de apartamentos en Bogota con el dataset apartamentos.csv.

Estructura del Repositorio:

- data/: Dataset con las variables necesarias para entrenar el modelo(apartamentos.csv) 
- notebooks/: Contiene el notebook taller2.ipynb el cual realiza todo el desarrollo del taller 
- README.md: Este archivo de contexto.
- Diccionario de datos - apartamentos.html 

Instrucciones de Uso:

1. Clonar el repositorio.
3. Ejecutar el notebook notebooks/taller2.ipynb

Resultados:

El modelo Random Forest fue el mejor modelo con un MAE (Error Medio Absoluto) de ~170 Millones y un R² de 0.88 indicando que el modelo explica el 88% de la variabilidad de los precios, lo cual es un rendimiento muy sólido para el sector inmobiliario..
